GetDaughterMenuItems ( $  type_id,
$  parent_id,
&$  search_items 
)

Возвращает в виде массива все дочерние элементы меню из имеющегося списка элементов $search_items для родительского элемента $parent_id, причём за каждым дочерним элементом массива сразу же следуют его дочерние элементы

Аргументы:
$type_id - id меню
$parent_id - id родительского элемента меню
&$search_items - ссылка на массив элементов меню, в котором ищутся дочерние
