Основная функция вывода Виджетов. Принимает в качестве парметров имя виджета (модуля), который необходимо подключить, в зависимости от параметров может не выводить ничего а просто проинициализировать модуль. вызов в шаблоне описывается в следующем виде: {widget name=catsubcat mode=element id=14} что означает вывести элемент номер 14 из модуля Текстовые страницы. Список стандартных параметров: name - задает имя подключаемого модуля; mode - функция, которую необходимо выполнить, обычно связанно с самим модулем, если не указана выполняется функция заданная в модуле по умолчанию. noprint - флаг (true|false) указывающий на необходимость разрешение/запрета вывода работы функции на экран.
