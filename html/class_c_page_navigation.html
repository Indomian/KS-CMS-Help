Подробное описание

Класс для работы с постраничной навигацией. Осуществляет обработку переданных данных или взятие их из строки запроса, так же возможна работа с потомками класса CObect для автоматической генерации страниц. При инициализации встраивается функция смарти, осуществляющая вывод постраничной навигации.
