Функция выполняет обработку одного элемента при создании списка элементов. Можно подменять в потомках, что позволяет получать дополнительные данные без дополнительного обхода массива

&$item указатель на запись которую требуется каким либо образом изменить. возвращает true если запись добавляется в результат или false если нет.
