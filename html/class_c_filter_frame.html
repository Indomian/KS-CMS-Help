Подробное описание

Класс выводит рамку для фильтрации списков. Автоматически определяет поля фильтрации по переданным в запросе данным, может возвращать значения массива для фильтрации при использовании функций потомков CObject::GetList().

Поля

$arFields

Список полей по которым осуществляется фильтрация. Содержит массив записей, структура записи такова: arRow=array( 'FIELD' -- имя поля для фильтрации, совпадает с именем поля в БД; 'TYPE' -- тип поля при выводе, допустимые значения: STRING,LIST,CHECKBOX; 'VALUE' -- установленное значение поля; 'VALUES' -- значения для полей типа список, массив где ключ - значение поля в БД, элемент - его название; 'METHOD' -- метод сравнения поля, по умолчанию - равно, также доступны >,<,~,>=,<= и ~ в качестве текстового поиска в поле; )
