Подробное описание

Родительский класс для модуля категорий, подкатегорий. В общем обычно абстрактен и сам по себе нигде не используется. Рекомендуется не применять этот класс а пользоваться его потомками CElement, CCategory. Начиная с версии 2.0 присутствует поддержка пользовательских полей.
